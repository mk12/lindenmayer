# Lindenmayer

Lindemayer is a web app that renders [fractal curves][1] generated by [L-systems][2].

[1]: https://en.wikipedia.org/wiki/Category:Fractal_curves
[2]: https://en.wikipedia.org/wiki/L-system

## Curves

- Koch snowflake
- Hilbert curve
- Peano curve
- ...
